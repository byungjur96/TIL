# Express

#### Express 설치하기

---

1. express 프레임워크를 설치한다.

   ```bash
   npm install -g express-generator
   ```

2. express를 `backend` 라는 이름의 폴더 안에 설치한다.

   ```bash
   express --view=pug backend
   ```

3. 해당 폴더 안으로 들어가서 웹서버를 실행한다.

   ```bash
   cd backend
   npm install npm start
   ```
